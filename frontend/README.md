# FeedPulse Frontend

Next.js App Router frontend for FeedPulse.

## Features

- Public feedback submission page
- Client-side validation
- Description character counter
- Admin dashboard with login, filters, sentiment badges, and status updates

## Environment Variable

Optional in `frontend/.env.local`:

- `NEXT_PUBLIC_API_URL=http://localhost:4000`

If not provided, the frontend defaults to `http://localhost:4000`.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

See root README for full project setup and assignment details.
